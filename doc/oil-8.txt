*oil-8.txt*           An Oil 6 based color scheme for |nvim|

===============================================================================
DESCRIPTION                                                 *oil-8-description*

Oil 8 is a |GUI|-only dark color scheme for |nvim| that was mainly based on the
Oil 6 color palette. Other color schemes inspirations includes Sopa de Mamaco,
Catppuccin and Bubblegum (see |oil-8-see-also|).

===============================================================================
REQUIREMENTS                                               *oil-8-requirements*

Oil 8 only requires the 'background' option set to `dark`.

===============================================================================
FEATURES                                                       *oil-8-features*

- Dark theme with base colors based on the Oil 6 color palette.
- Colors distributed throughout the spectrum.
- Similar colors have similar meanings.
- Integration with |nvim-treesitter| and |neo-tree| (see |oil-8-integration|).

===============================================================================
USAGE                                                             *oil-8-usage*

To load the color scheme with its default configuration, use the traditional
|:colorscheme| command, as shown below.
>vim
  :colorscheme oil-8
<
To load the color scheme with custom configuration, use the `setup` function
(|oil-8.setup()|), as exemplified in the following Lua code snippet.
>lua
  require'oil-8'.setup{
    -- Terminal color support is disabled by default. Set it to `true` to
    -- enable.
    terminal_colors = false,

    -- By default, most plugins have integration enabled. To disable all plugin
    -- integration, set the `integration` field to `false`.
    integration = {
      neo_tree = true, -- Integration for Neo-tree.
      treesitter = true, -- Integration for nvim-treesitter.
    }
  }
<
===============================================================================
INTEGRATION                                                 *oil-8-integration*

Oil 8 provides seamlessly theme integration for other plugins of the Neovim's
ecosystem. The integration is configured via the `integration` field
(|oil-8-config.integration|) of the configuration table that may be passed as
argument for the `setup` function (|oil-8.setup()|).

The supported plugins are listed below.

- Illuminate (|illuminate.txt|)
- Indent Blankline (|indent-blankline.txt|)
- Leap (|leap.txt|)
- Mini (|mini.txt|)
  - Indentscope (|mini.indentscope|)
- Neo-tree (|neo-tree.txt|)
- nvim-treesitter (|nvim-treesitter|)

===============================================================================
API                                                                 *oil-8-api*

The following subsections cover each module/submodule composing the Lua API of
Oil 8.

OIL-8 ROOT MODULE

`oil-8`                                                                 *oil-8*
  Table serving as the main module for this plugin.

`oil-8`.setup({?config})                                        *oil-8.setup()*
  Function for loading the color scheme according to optional {?config} or
  their defaults. If no {?config} would be passed to this function, it's
  recommended to use `:colorscheme oil-8` instead.

  The documentation for {?config} can be found at |oil-8-config|.

OIL-8.PALETTE SUBMODULE

`oil-8.palette`                                                 *oil-8.palette*
  Table serving as the module containing all the colors used by this color
  scheme. Each key stands for the color name, with its corresponding value
  being a string representing a hex triplet for the color.

  The comprehensive list of all the colors can be found at |oil-8-palette|.

===============================================================================
CONFIGURATION                                                    *oil-8-config*

Oil 8 is configured via the configuration table passed as argument to the
`setup` function (|oil-8.setup()|). The fields of that table are listed and
explained below.

`terminal_colors`                                *oil-8-config.terminal_colors*
  Boolean controlling whether Oil 8 should set terminal color variables
  (see |terminal-config|). By default, it's set to `false`.

`integration`                                        *oil-8-config.integration*
  Key-value table controlling which (and how) plugins should be integrated by
  Oil 8. Each key represents the plugin name in snake case, with the
  corresponding value typically being a boolean enabling/disabling the
  integration for that plugin.

  NOTE: Setting the `integration` field to `false` will completely disable
        plugin integration functionality.

  In most cases, plugin integration consists of setting highlight groups
  specific to the plugin's functionality.

  The keys of the `integration` table (i.e., the plugins supported by Oil 8),
  and their corresponding values are listed below.

  `illuminate`                            *oil-8-config.integration.illuminate*
    Boolean controlling whether to enable integration for Illuminate
    (|illuminate.txt|). By default, this plugin is integrated by Oil 8.

  `indent_blankline`                *oil-8-config.integration.indent_blankline*
    Boolean controlling whether to enable integration for Indent Blankline
    (|indent-blankline.txt|). By default, this plugin is integrated by Oil 8.

  `leap`                                        *oil-8-config.integration.leap*
    Boolean controlling whether to enable integration for Leap (|leap.txt|). By
    default, this plugin is integrated by Oil 8.

  `mini`                                        *oil-8-config.integration.mini*
    Key-value table controlling Mini (|mini.nvim|) modules that should be
    integrated by Oil 8. Setting the `integration.mini` field to `false` will
    completely disable plugin integration functionality for Mini.

    `indentscope`                   *oil-8-config.integration.mini.indentscope*
      Boolean controlling whether to enable integration for Indentscope
      (|mini.indentscope|). By default, this plugin is integrated by Oil 8.

  `neo_tree`                                *oil-8-config.integration.neo_tree*
    Boolean controlling whether to enable integration for Neo-tree
    (|neo-tree.txt|). By default, this plugin is integrated by Oil 8.

  `treesitter`                            *oil-8-config.integration.treesitter*
    Boolean controlling whether to enable integration for nvim-treesitter
    (|nvim-treesitter|). By default, this plugin is integrated by Oil 8.
